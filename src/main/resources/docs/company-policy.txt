公司年假政策：
入职满1年的员工享有5天年假。
入职满3年的员工享有10天年假。
入职满5年的员工享有15天年假。
年假可以跨年使用，但最多累积到次年3月31日。
请假需要提前3个工作日在OA系统提交申请。
--
公司报销政策：
出差交通费用全额报销，需提供发票。
餐饮补贴标准：一线城市每天200元，二线城市每天150元。
住宿标准：一线城市每晚500元以内，二线城市每晚350元以内。
报销需在出差结束后7个工作日内提交。
--
公司技术栈：
后端主要使用 Java 17 + Spring Boot 4。
前端使用 React + TypeScript。
数据库使用 MySQL 和 Redis。
消息队列使用 RocketMQ。
部署在阿里云 Kubernetes 集群上。
代码管理使用 GitLab，CI/CD 使用 Jenkins。
